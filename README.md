# Install Spice from Brew

## Pre-requisites

Noto: when using linux homebrew, install build-essentials or equivalent package.

## Install `spice`
```
brew install spiceai/spiceai/spice
```

## Install `spiced`
```
brew install spiceai/spiceai/spiced
```

## Update version

```
./update-version <version>
```
